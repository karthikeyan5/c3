# C3 STT (speech-to-text) Python dependencies.
#
# Installed into the dedicated venv at ~/.config/c3/stt-venv by setup-venv.sh.
# C3's STT plugin auto-detects that venv (~/.config/c3/stt-venv/bin/python) and
# runs the handler under it, so these deps work even when the system python3 is
# externally-managed (PEP 668) and cannot pip-install them.
#
# SYSTEM dependency (NOT pip): ffprobe, from ffmpeg — install via your OS package
# manager (e.g. `sudo pacman -S ffmpeg`, `apt install ffmpeg`). Used to detect
# audio duration; STT degrades gracefully (REST-then-batch) if it is absent.

# NO Python PyPI dependencies. Both the REST (≤30s) and batch (>30s) Sarvam
# flows now use only the stdlib (urllib/json/os/time). The `sarvamai` library
# was removed (2026-06-22) — its sole use was the >30s batch path, since
# reimplemented over native HTTP in providers/sarvam-saaras-v3.py.
#
# This file is intentionally dependency-free; it is kept so setup-venv.sh and
# the docs have a stable target and future deps have a home.
